The **`[logging]`** section configures application logging. **Static** — the logger is initialized once at startup.

The **`level`** can be overridden with the **`RUST_LOG`** environment variable.

<Tabs syncKey="huginn-config">
  <TabItem label="YAML">

```yaml
logging:
  level: "info"
  show_target: false
```

  </TabItem>
  <TabItem label="TOML">

```toml
[logging]
level = "info"
show_target = false
```

  </TabItem>
</Tabs>

| Key | Default | Description |
| --- | --- | --- |
| `level` | `"info"` | `trace`, `debug`, `info`, `warn`, or `error`. |
| `show_target` | `false` | Include the Rust module path in log lines (useful for debugging). |
