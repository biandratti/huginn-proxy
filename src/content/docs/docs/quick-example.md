The repository’s [**examples**](https://github.com/biandratti/huginn-proxy/tree/master/examples) directory documents an end-to-end stack (Compose, TLS, sample configs, backends, telemetry, optional eBPF, and how to exercise it). Follow the README there to run it with the **latest published Docker image** from the registry (`ghcr.io`, tags and `docker compose` commands are listed alongside the files).
